Fetch Twitter Tweets and Instagram Photos into a beautiful gallery

== Description ==

As simple as the name is.

This plugin gives you a widget to display Twitter Tweets and Instagram photos in gallery layout.

You enter your Twitter and Instagram accout name. (For Twitter, you need to register and provide API keys).

That's it.

Put the widget anywhere.

You can also use Shortcode to display the gallery

[ct_social_feed twitter_name="" twitter_consumer_secret="" twitter_access_token_secret="" twitter_access_token="" twitter_consumer_key="" instagram_name=""]

Twitter API Keys can be obtained from here: <a href="https://dev.twitter.com/apps/new" target="_blank">Twitter App</a>

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets to use the widget
